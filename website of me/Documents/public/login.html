User Login

Users can login here with email or mobile. Admin access is separate and opens from the small header icon with a secure two-step login.

Reset a user password by entering the same full name and contact used during signup, then choose a new password.

admin

Separate Admin Login

Use the small `AD` icon in the header to login as admin. Step 1 verifies the first password, then Step 2 unlocks the second admin password field.

If you forget admin access, click `Forgot Password?` in the admin modal and enter both recovery birthdays before resetting the first and second admin passwords.
